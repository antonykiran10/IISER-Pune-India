About DeleMa Detect

This is a demo server that is able to predict if a blood smear image is infected with Plasmodium falciparum or not for Malaria detection.

This is a part of iGEM IISER Pune '20 Project. More details can be found on our Wiki Page.

This Demo Web App is a part of DeleMa Detect, a deep learning framework for malaria diagnosis. Information about the Software and the usage protocol can be found on the DeleMa Detect Wiki Page.

The Server Code can be found on iGEM IISER Pune's GitHub Repo.
